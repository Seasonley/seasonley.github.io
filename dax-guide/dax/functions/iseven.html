ISEVEN DAX Function (Information)

Syntax  |  Return values  |  Examples  |  Related  

Returns TRUE if number is even, or FALSE if number is odd.

Syntax

ISEVEN ( <Number> )
Parameter Attributes Description
Number

The value to test. If number is not an integer, it is truncated.

Return values

Scalar A single boolean value.

Returns TRUE if Number is even, or FALSE if number is odd.

» 1 related function

Examples

--  ISEVEN checks that a number is EVEN,
--  ISODD checks that a number is ODD
EVALUATE
    {
       ( "ISEVEN ( 10 )", ISEVEN ( 10 ) ),
       ( "ISODD  ( 10 )", ISODD  ( 10 ) ),
       ( "ISEVEN ( 11 )", ISEVEN ( 11 ) ),
       ( "ISODD  ( 11 )", ISODD  ( 11 ) )
    }
Value1 Value2
ISEVEN ( 10 ) true
ISODD ( 10 ) false
ISEVEN ( 11 ) false
ISODD ( 11 ) true
