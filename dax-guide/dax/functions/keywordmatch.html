KEYWORDMATCH DAX Function Deprecated

Syntax  |  Return values  |  Remarks  

Returns TRUE if there is a match between the MatchExpression and Text.

Syntax

KEYWORDMATCH ( <MatchExpression>, <Text> )
Parameter Attributes Description
MatchExpression

MatchExpression to be evaluated against Text.
Text

Text to be evaluated against MatchExpression.

Return values

Scalar A single boolean value.

Returns TRUE if there is a match between the MatchExpression and Text.

Remarks

This function has been never released. It was present in some version of Excel.
