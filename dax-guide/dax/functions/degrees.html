DEGREES DAX Function (Math and Trig)

Syntax  |  Return values  |  Examples  |  Related  

Converts radians into degrees.

Syntax

DEGREES ( <Number> )
Parameter Attributes Description
Number

The angle in radians that you want to convert.

Return values

Scalar A single decimal value.

» 1 related function

Examples

--  RADIANS and DEGREES convert one unit of measure
--  in the other one, for angles.
DEFINE 
    VAR Vals = GENERATESERIES ( -PI(), PI(), PI()/4 )
EVALUATE
    ADDCOLUMNS ( 
        Vals,
        "DEGREES",   DEGREES( [Value] ),
        "RADIANS",   RADIANS ( DEGREES ( [Value] ) )
    )
