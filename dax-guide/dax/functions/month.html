MONTH DAX Function (Date and Time)

Syntax  |  Return values  |  Remarks  |  Examples  |  Articles  |  Related  

Returns a number from 1 (January) to 12 (December) representing the month.

Syntax

MONTH ( <Date> )
Parameter Attributes Description
Date

A date in datetime format.

Return values

Scalar A single integer value.

An integer number indicating the number of the month.

Remarks

If the argument is a string, it is translated into a datetime value using the same rules applied by the DATEVALUE function.

Examples

--  Extract date parts from a date
EVALUATE
ADDCOLUMNS (
    TOPN ( 10, VALUES ( 'Date'[Date] ), 'Date'[Date], ASC ),
    "Year",    YEAR    ( 'Date'[Date] ),     
    "Quarter", QUARTER ( 'Date'[Date] ),
    "Month",   MONTH   ( 'Date'[Date] ),    
    "Day",     DAY     ( 'Date'[Date] )         
)
ORDER BY 'Date'[Date]
